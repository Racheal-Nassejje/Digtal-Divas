# Digital Divas — Restaurant Menu & Ordering Page

## Project Description
A web-based system with HTML,CSS, JAVASCRIPT that transitions a restaurant 
from a manual system to an automated system.
The system allows customers to view the menu 
and place orders online.

## Project Type
Business

## Problem Statement
Majority of restaurant currently uses a manual system where 
customers view physical menus and place orders by hand. 
This causes delays, errors and poor customer experience.
This system solves that by automating the process online.

## Primary users(Uses the system themselves)
- Customers
  (Directly use website to view the digital menu and place orders online)

## Secondary Users(affected by what primary user does)
- Waiters/Staff
  (receive and process customer orders)
- Restaurant Manager
  (Monitors orders and updates the menu)

├── order.html       → Ordering Page

## Page Descriptions

### index.html — Homepage
- Restaurant Name and Logo
- Navigation Bar (Home | Menu | Order)
- Welcome Message
- About the Restaurant
- Footer

### menu.html — Menu Page
- Navigation Bar
- Menu Categories (Starters | Main Course | Drinks | Desserts)
- Food Items with Names and Prices
- Footer

### order.html — Ordering Page
- Navigation Bar
- Order Form with Customer Name, Phone Number,
  Select Food Item, Quantity and Submit Button
- Footer

### style.css — Styling
- Colors and Fonts
- Navigation styling
- Cards for menu items
- Form styling
- Button effects
- Footer styling

### script.js — JavaScript
- Navigation menu toggle
- Menu category filter
- Order form validation
- Order confirmation message
- Total price calculator

## How Pages Connect
- index.html links to menu.html and order.html
- style.css styles all pages
- script.js makes all pages interactive.
- CSS Styling (style.css)

The style.css file controls the visual design of the website.
It includes:

Website color scheme
Font styling
Navigation bar styling
Card layout for menu items
Order form styling
Button hover effects
Footer styling

The pages are linked together as follows:
index.html → menu.html
index.html → order.html
menu.html → order.html
The style.css file applies styling across all pages, while script.js provides interactive features.
